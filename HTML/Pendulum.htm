Description

Pendulum combines two free apps to collect and analyze accelerometer data.

File data collected by 'Physics Toolbox Accelerometer' app is transformed to Excel spreadsheet 'Pendulum/copy.xls' using the template spreadsheet 'Pendulum/input.xls'. The selected accelerometer data is analyzed to determine the times of peak acceleration to allow calculation of pendulum period and gravity.

The template file can be edited to perform other analysis functions but note that columns 1-8 will be over-written by data.

The following are recommended apps for collection and analysis.

Physics Toolbox Accelerometer
https://play.google.com/store/apps/details?id=com.chrystianvieyra.android.physicstoolboxaccelerometer
